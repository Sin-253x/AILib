# ======================== 代码解释 ========================
# 1. 整体功能：
#    固定 API 容器运行时依赖版本，提升 Docker 构建的可复现性。
# 2. 关键部分拆解：
#    - 直接依赖：对应 pyproject.toml 中 FastAPI、LangChain、DeepSeek、数据库和文档解析依赖。
#    - 传递依赖：锁定本地测试已验证过的依赖树，减少 pip resolver 在容器内反复搜索。
#    - 不包含开发依赖：pytest、ruff 等只用于本地验证，不进入生产镜像。
# 3. 重要概念与库：
#    - requirements lock：用固定版本约束镜像构建结果，降低上游包更新带来的漂移。
#    - --no-deps 安装：Dockerfile 会按本文件安装完整依赖树，然后单独安装项目包。
# 4. 潜在问题与改进建议：
#    - 如果升级 LangChain、FastAPI 或 DeepSeek SDK，需要重新运行测试并同步刷新本文件。
# 5. 修改指南：
#    - 新增运行时依赖时，应同时更新 pyproject.toml 和本文件。
# ========================================================
